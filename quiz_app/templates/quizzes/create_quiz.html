The system supports bulk import of questions from .xlsx files

  • Column A: Question content
  • Columns B, C, D, E: Answers 1, 2, 3, 4
  • Column F: Correct answer position (Enter number from 1 to 4)
  • Column G: Explanation (Can be left blank)

Note: Leave row 1 blank (header row). The system will start reading from row 2.
